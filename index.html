git is a DAG

branches are just pointers

 > git branch -a -v

Branches

List:

 > git branch -a -v

Create:

 > git branch branch-name [points-to]

Index (to be committed)

Show:

 > git diff --cached

Add to:

 > git add file-name

Remove from:

 > git reset file-name

Working Tree

Show:

 > git diff [--stat]

Discard local changes:

 > git checkout file-name

Patch mode

Add to index:

 > git add -p

Remove from index:

 > git reset -p

Discard local changes:

 > git checkout -p

Switch branches

 > git checkout gh-pages
 > git checkout master
