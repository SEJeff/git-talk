Branches

List:

git branch -a -v

Create:

git branch branch-name [points-to]

Index (to be committed)

Show:

git diff --cached

Add to:

git add file-name

Remove from:

git reset file-name

Working Tree

Show:

git diff [--stat]

Discard local changes:

git checkout file-name

Patch mode

Add to index:

git add -p

Remove from index:

git reset -p

Discard local changes:

git checkout -p

Stash

Show:

git stash list

Save:

git stash save "Some message"

git stash

Apply:

git stash apply stash@{2}
# OR restore and drop most recent stash
git stash pop

feature

# list all unmerged feature-branches
feature list
# list all feature-branches
feature list -v
       
# creates a new branch of master
feature start name-of-feature
# does git-checkout
feature switch name-of-feature
# merges back into master
feature finish name-of-feature
       
# lists all stashes on the current branch
feature stashes
